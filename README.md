# SpringBootScan

SpringBoot 漏洞扫描

## 系统
```
linux:
  SpringScan_linux_386
  SpringScan_linux_amd64
mac:
  SpringScan_darwin_386
  SpringScan_darwin_amd64
windows:
  SpringScan_windows_386.exe
  SpringScan_windows_amd64.exe  
```
